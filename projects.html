Teams Sales RAG Agent

Konceptskiss

AI-driven säljassistent för Microsoft Teams med Enterprise RAG

⚠ Detta är en arkitektur- och idéskiss, inte ett färdigt projekt. Syftet är att demonstrera hur en sådan lösning skulle kunna designas och integreras i ett Enterprise M365-ekosystem.

Problemet

Säljteam spenderar för mycket tid på att söka i interna system efter produktinfo, prislistor och kundhistorik. Informationen finns men är utspridd och svår att nå snabbt i en kundkonversation.

Lösningen

En RAG-agent integrerad direkt i Microsoft Teams som låter säljare chatta säkert med intern företagsdata. Byggd med Semantic Kernel och Azure AI Search – all data stannar inom företagets Microsoft 365-tenant.

Semantic Kernel Azure AI Search Microsoft Teams RAG Python

GitHub Issue Auto-Agent

Pollar GitHub-repos och låter AI lösa issues automatiskt

Problemet

Repetitiva issues och boilerplate-uppgifter tar oproportionerligt mycket tid från kärnutveckling.

Lösningen

En daemon pollar repos var 15:e minut, identifierar issues med specifik label och startar en AI-agent som analyserar, implementerar, testar och öppnar en PR – helt utan manuell intervention.

OpenClaw Sub-agents

Hierarkiskt multi-agent-system med parallell exekvering

Problemet

Enkla LLM-anrop kan inte hantera komplexa uppgifter som kräver specialisering och parallellism. Sekventiella anrop skalas dåligt kostnadsmässigt.

Lösningen

Manager/worker-mönster där en orchestrator bryter ner uppgifter och delegerar till specialiserade underagenter i parallell exekvering. Resultaten aggregeras och valideras av managern.

Lokal RAG

Retrieval-Augmented Generation-pipeline utan molntjänster

Problemet

Känsliga interna dokument kan inte processas via externa AI-API:er utan att bryta mot dataskyddskrav eller interna säkerhetspolicies.

Lösningen

Fullständig RAG-pipeline som kör lokalt: dokumentingestion, chunking, embedding via lokal modell, vector store och query-loop mot lokal LLM. Ingen data lämnar miljön.

MCP-brygga & OpenClaw Bot

Personlig AI-infrastruktur med långtidsminne och lokal RAG via MCP

Problemet

Kommersiella AI-assistenter saknar persistent minne, känner inte till din kontext och kan inte agera autonomt i din miljö. Varje session börjar från noll.

Lösningen

En OpenClaw-baserad autonom agent med en MCP-brygga (Model Context Protocol) som kopplar till en lokal RAG-hjärna. Agenten har långtidsminne, server-access och kan utföra uppgifter självständigt – ett levande exempel på hur AI faktiskt automatiserar vardagsarbetet.
